# MoneyMemo

一款现代化的记账应用，支持多币种、可自定义小数位数，并提供直观的收支统计和账本管理功能。

---

## 功能特点

### 账本管理
- 添加、编辑、删除账目
- 按时间范围（年/月/周）查看收支情况
- 分类管理支出和收入

### 收支统计
- 显示支出、收入和结余
- 支持自定义小数位数
- 多币种支持，自动显示对应符号

### 多币种与小数位数设置
- 支持 CNY、USD、HKD
- 可在设置中修改小数位数（0、1、2 位）
- 实时更新账本显示

### 界面与交互
- 深色模式切换
- 响应式布局，适配各种屏幕
- SwiftUI 动画效果提升用户体验

### 其他功能
- 关于页面
- 重置信息功能

---

## 技术栈
- **前端**：SwiftUI
- **后端**：Supabase + PostgREST
- **数据存储**：Supabase 数据库
