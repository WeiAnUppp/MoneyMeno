# MoneyMemo

一款现代化的记账应用，支持多币种、可自定义小数位数，并结合 GLM 图像识别能力，实现更高效、智能的记账体验。

---

## 功能特点

### 账本管理
- 添加、编辑、删除账目  
- 按时间范围（年 / 月 / 周）查看收支情况  
- 分类管理支出和收入  
- 最近账单快速查看  

### 智能记账（GLM 图像识别）
- 支持通过 **选择图片** 进行记账  
- 基于 **GLM 图像识别（OCR）** 自动识别消费截图  
- 可自动提取：
  - 金额  
  - 日期  
  - 消费类型（如餐饮、交通、购物等）  
- 识别结果支持手动校正后保存  
- 显著降低手动输入成本，提高记账效率  

### 收支统计
- 显示支出、收入和结余  
- 按分类统计收支占比  
- 支持自定义小数位数显示  
- 数据随设置变更实时更新  

### 多币种与小数位数设置
- 支持多种货币：
  - CNY（￥）
  - USD（$）
  - HKD（HK$）
- 自动显示对应货币符号  
- 可设置小数位数（0 / 1 / 2 位）  
- 设置修改后，账本与统计页面即时生效  

### 界面与交互
- 深色模式 / 浅色模式切换  
- 使用 SwiftUI 动画与过渡效果，提升整体体验  

### 其他功能
- 关于页面
- 数据重置功能  

---

## 技术栈

- **前端**：SwiftUI  
- **后端**：Supabase + PostgREST  
- **数据存储**：PostgreSQL
- **智能识别**：GLM-4.6V 图像识别（OCR）
